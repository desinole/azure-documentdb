# Code Samples

This folder contains code examples demonstrating Azure DocumentDB functionality.

## Contents

Code samples covering:
- Connection and authentication
- Basic CRUD operations (Create, Read, Update, Delete)
- Querying documents
- Indexing strategies
- Performance optimization
- Best practices

## Structure

Each sample includes:
- Complete, runnable code
- Comments explaining key concepts
- Prerequisites and setup instructions
- Expected output or results

## Prerequisites

- Azure DocumentDB instance (local or cloud)
- Required SDKs and dependencies (specified in each sample)

## Getting Started

Refer to individual sample folders for specific setup and execution instructions.
